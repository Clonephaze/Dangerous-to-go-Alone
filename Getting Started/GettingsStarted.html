Installation

Establishing Cemu Install

Make sure you have a valid and working copy of the Breath of the Wild. We cannot legaly tell you how to do this.

Download and install Cemu, link is to the right.

Open the Cemu application, click File -> Install game title, update or DLC. Open the folder containing the game, open the Meta folder and select meta.xml.

Launch your game and make sure it works before installing any mods.

Installing and setting up BCML

Make sure you have the lastest x64 Visual C++ installed, link is to the right.

Install python 3.8.10, link is to the right.

Open Command Prompt, press Win+R and enter "cmd".

Type "pip install BCML" and press enter. BCML will be installed for you.

Open BCML by typing "bcml" and press enter.

BCML will run through a setup wizard with you, follow its instructions.

Downloading and Installing Required Mods

Download the Collision Actors mod, required for DtgA, link is to the right.

Download the DtgA modules Reborn, Reforged, and Reanimated. You can find the download links on our Home page.

Optionally, you can download the RePopulated (DtgA's first DLC module!) mod file as well if you would like that content too.

Open BCML if it's not already open, and click the plus button in the bottom right hand corner to install all downloaded mods.

In BCML the highest mod has the highest priority. Ensure Collision Actors is higher than DtgA mods.

You can now launch your game from BCML and play! Choose "Play DtgA the way Zoey intended" in the main menu to get the full experience.

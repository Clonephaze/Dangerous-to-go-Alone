Getting Started with the Mod

To get started playing with this mod, you will need to follow these steps:

  1. Make sure you have a copy of the Breath of the Wild.
  2. Download and install Cemu, a Wii U emulator.
  3. Download the DtgA mod files, you will need Reborn Reforged AND ReAnimated.
  4. Download the Collision actors mod.
  5. Install BCML (Breath of the Wild Cemu Mod Loader) to manage and load the mod.
  6. Open BCML and run through its configuration process. When complete click the plus button to install each individual mod file. Ensure Collison actors is placed above all DtgA files.
  7. Use BCML to launch Cemu and configure your graphic pack settings. Turn on "extended memory" and "fps++", these are required for the mod to work properly.
  8. Launch the game through BCML, and enjoy your journey!
